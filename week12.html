What did you do last week?

This week I played around with React and checked if it is something I can add to my website. React is a popular JavaScript library for building user interfaces, particularly single-page applications where you need a responsive and dynamic user experience. From my research, the most common approach is to use Django as the backend, providing an API (using the Django REST Framework), and React as the frontend for the user interface. The React application makes HTTP requests to the Django backend to fetch or manipulate data. It is pretty cool and I was able to make some nice interfaces, but I didn't integrate it with my website. This was more just to see how it works since it is very popular with web designers, including some of the stuff my team supports at work. Our onboarding application uses React for example.
